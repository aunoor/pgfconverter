Программа позволяет экспортировать и импортировать точки из GPX файлов 
в файл "Избранное" ПроГород.

Основные принципы работы с программой:

Кнопка "Открыть файл" позволяет загрузить список точек из gpx или favorites.dat файла.
Для добавления новых точек в уже открытый список служит кнопка "Добавить из файла".
Для выгрузки отмеченных точек в формат gpx служит кнопка "Сохранить в gpx", а для выгрузки в формат файла
"Избранное" кнопка "Сохранить в ПроГород".
По двойному щелчку на записи с точкой вызывается окно редактирования наименования и описания точки.

При экспорте в favorites.dat не экспортируются следующие поля:
- адрес
- телефон
- тип иконки
- тип места (дом, офис)

При импорте из GPX и favorites.dat импортируются за исключением 
координат только поля описания и наименования точки.

ПроГород поддерживает только 5 значимых знаков после запятой в сохраняемых координатах.
Из-за этого, преобразование из gpx может приводить к потерям точности, но т.к. погрешность
GPS до 10 метров, несколько секунд большой разницы не играют.

Программа распостраняется As Is, и автор не несет ответственность за возможные
последствия ее применения.

Исходные коды программы можно найти на http://code.google.com/p/pgfconverter/
и распостраняются под лицензией BSD.
